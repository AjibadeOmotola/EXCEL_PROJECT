# Excel Projects

This folder contains all my Excel projects demonstrating foundational and intermediate Excel skills, data cleaning, and dashboard creation.

## Skills & Types of Projects Included

- Basic Excel formulas: SUM, AVERAGE, MAX, MIN, IF, AND, OR  
- Lookup and automation: VLOOKUP  
- Data summarization and reporting: PivotTables  
- Conditional formatting applications  
- Data cleaning and preparation: handling missing values, removing duplicates, formatting datasets  
- Dashboard creation and data visualization with charts and tables  
- HR and business-related Excel projects  

**Note:** Multiple files may cover more than one skill or function. Each file is named to reflect its purpose.

## Tools Used

- **Microsoft Excel**  
- **Excel Functions & Formulas** (SUM, AVERAGE, MAX, MIN, IF, AND, OR, VLOOKUP)  
- **PivotTables & Charts**  
- **Conditional Formatting**  
- **Data Cleaning Techniques**  

## How to Explore

- Open any Excel file to see the formulas, functions, and analysis applied.  
- Each file demonstrates one or more Excel skills in practice.
